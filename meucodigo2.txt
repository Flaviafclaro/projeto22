git basico 2

Sequência
1 - clique direito na pasta e git bash here
2 - git init + enter
3 - git status + enter

============================
Branch
git branch           //vai exibir as branchs disponiveis
git staging          //cria uma branch de nome staging
git chackout staging //trabalhando na branch staging
git checkout -b nomedabranchNova nomedabranchBase
               //criando uma branch nova baseada em outra
============================
Merge
git checkout master    //indo para a branch para onde vai o merge
git merge staging      //staging é o nome da branch que eu fiz
git push
============================
Pull
git pull  //trazendo a ultima versão antes de dar push, caso             alguém tenha feito outras atualizações no caso de             trabalho em grupo.
============================
Ignore
touch .gitignore   //criando o arquivo ignore

abrir este arquivo como bloco de notas e incluir o caminho dos arquivos a sere ignorados, o gitignore deve ser adicionado com o git add .
